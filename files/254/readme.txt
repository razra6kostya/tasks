2.54 Напишите программу, которая получает через аргументы командной строки имена нескольких (произвольного количества, не менее одного) существующих бинарных файлов и одного текстового файла, который программа должна создать во время работы (его имя задаётся последним). 
Проанализируйте каждый из заданных бинарных файлов, запишите в текстовый файл с заданным именем строки (по одной для каждого проанализированного файла), содержащие имя файла, общее количество чисел в нём, наименьшее и наибольшее из чисел, содержащихся в файле.
Для отладки вашего решения напишите вспомогательную программу, которая формирует бинарный файл нужного вам формата, состоящий из чисел.
