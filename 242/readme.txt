Напишите программу, которая читает из стандартного потока ввода строки, состоящие из слов (слова могут разделяться произвольными группами пробельных символов), и в ответ на каждую прочитанную строку печатает слова из этой строки В ОБРАТНОМ ПОРЯДКЕ; например, в ответ на фразу "Hympty Dumpty sat on a wall" должно быть напечатано "wall a on sat Dumpty Hympty". Вводить ограничения на длины строк, слов и ДР.  нельзя; в частности, недопустимо использовать тип string для хранения читаемых строк и/или отдельных слов. Вся выделенная динамическая память должна быть корректно освобождена сразу после обработки очередной строки.
